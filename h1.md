<h1>*c) "Hei maailma" Ansiblella ja testaus SSH:n yli:*<h1/>
  
Jälleen poistuin SSH yhteydestö localhostista ja jatkoin asentamalla Ansiblen.   
Käytin tällä kertaa dokumentaatiosta löytyvää; **sudo apt-get install ansible micro bash-completion tree**-komentoa, tällä kertaa asensin samassa yhteydessä myös micro tekstieditorin.  
Loin ansiblea varten hakemiston **mkdir ansible/** ja siirryin juuriluotuun hakemistoon; **cd ansible**.  
Tämän jälkeen loin listan Hosteista/Isäntäkoneista kyseiseen hakemistoon ja lisäsin localhostin listaan,  
ansiblen yhteydessä asennetulla micro tekstieditorilla; **micro host.ini** ja tarkistin että lista oli tallentunut; **cat host.ini**   
Testasin Ansiblen toimivuutta komennolla: **ansible all -i hosts.ini -m ping**, joka palautti Varoitusviestin Python kirjaston versiosta, ilmoituksen onnistuneesta yhteydestä, muutoksista ja hassunhauskan ***pong***-vastauksen ping pyyntöön.  
<img width="1892" height="657" alt="kuva" src="https://github.com/user-attachments/assets/35d11366-7325-4cfa-a146-b94352cd7f7a" />  
Kokeilin tämän jälkeen dokumentaatiosta löytyviä ohjeita kuinka Python varoitukset saisi pysäytettyä  
Lisäsin: **[all:vars]ansible_python_interpreter=/usr/bin/python3** host.ini tiedostoon, jonka jälkeen varoitustekstejä ei enään näkynyt.
